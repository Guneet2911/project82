Project on console

Visual Studio Code is an offline IDE that allows us to code without using internet. We can code anytime and anywhere on Visual Studio.

How to setup a project on VS code:

Step1: Create a folder on the computer.

Step2: Upload it on Visual Studio.

Step3: Create three files:

"index.html , style.css , main.js".

A console shows us what is happening in our application.To show text on the console, we use Javascript.
